# js101-scoreboard-app
Module 01 of the JS101 Practice Series.A project-based learning resource for mastering JavaScript fundamentals. Built with HTML5, CSS3, and Vanilla JS logic.

## What you'll learn
- DOM manipulation
- Event handling
- Variables
- JS conditions

## Why This Matters?
I developed this project to help my students move past basic syntax. By focusing on a scoreboard, they learn how to manage Application State (the score) and synchronize it with the User Interface (the display)—a fundamental skill for modern web development.

## How to use
1. Go to the 'starter' folder
2. Try to complete the project
3. If you get stuck, check the 'solution' folder
